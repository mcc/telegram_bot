## Instructions to run:

- Change working directory to project directory: `cd /path/to/telegram_bot`
- Add bot token to config.yaml file.
- Install the requirements: `$pip3 install -r requirements.txt`
- Run the bot: `$python3 bot.py`
- Stop the bot: `ctrl + c`
